# 🔥 Algerian Forest Fire Prediction

This project predicts Forest Fire Weather Index (FWI) using Machine Learning regression models.

## 📊 Dataset
Algerian Forest Fires Dataset

## ⚙️ Steps Performed
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Train Test Split
- Standardization
- Model Training
- Model Evaluation

## 🤖 Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- RidgeCV
- LassoCV
- ElasticNet

## 📈 Evaluation Metrics
- R2 Score
- Mean Absolute Error
- Mean Squared Error

## 🛠️ Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
